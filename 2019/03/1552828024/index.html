Es6--箭头函数

发表于 2019-03-13 | 分类于 javaScript
字数统计: 466 | 阅读时长 ≈ 2

ES6可以使用“箭头”（=>）定义函数，注意是函数，不要使用这种方式定义类（构造器）。

一、语法

具有一个参数的简单函数

　　

没有参数的需要用在箭头前加上小括号

　　

多个参数需要用到小括号，参数间逗号间隔，例如两个数字相加

　　

函数体多条语句需要用到大括号

　　

返回对象时需要用小括号包起来，因为大括号被占用解释为代码块了

　　

直接作为事件handler

　　

作为数组排序回调

　　

二、注意点

typeof运算符和普通的function一样

　　

instanceof也返回true，表明也是Function的实例

　　

this固定，不再善变

这个很有用，再不用写me，self，_this了，或者bind。

箭头函数不能用new

　　

不能使用argument

　　

对于5，在Firefox36里测试是可以输出55的，貌似并没有这个限制
